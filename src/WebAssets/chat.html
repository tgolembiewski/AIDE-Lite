Data Processing Consent

Before using AIDE Lite, please understand how your data is processed:

  • App model data (entity names, attributes, microflow structures, page names) is sent to Anthropic's Claude API to provide context for AI responses.
  • Your chat messages are sent to Anthropic for processing and response generation.
  • Anthropic is a US-based company. Your data is transmitted via HTTPS and processed on their infrastructure.
  • Conversation history is stored locally on your machine with DPAPI encryption. Up to 50 conversations are retained.
  • Your API key is encrypted with DPAPI and stored locally. It is only used to authenticate with Anthropic's API.

You can control the amount of data sent via the Context Depth setting (Summary or None sends less data).

For more details, see the Privacy Notice (lock icon in the header) or review Anthropic's Privacy Policy.

Privacy Notice

Data Collected

  • App model context: Entity names, attribute names/types, association mappings, microflow names/parameters/activities, page names, and enumeration values from your Mendix project.
  • Chat messages: Text you type in the chat input.
  • Tool results: Data returned by tool calls (model queries, microflow operations).

How Data is Processed

  • Data is sent via HTTPS to Anthropic's Claude API (api.anthropic.com) for AI-powered responses.
  • Anthropic processes data on US-based infrastructure.
  • Refer to Anthropic's Privacy Policy for their data handling practices.

Data Stored Locally

  • API key: DPAPI-encrypted in %APPDATA%\AideLite\config.json
  • Conversation history: DPAPI-encrypted in %APPDATA%\AideLite\history\ (max 50 conversations)
  • Debug log: %APPDATA%\AideLite\debug.log (rotated at 5MB, no message content)

Your Rights

  • Delete conversations: Use the History panel to delete individual conversations.
  • Export data: Use the Export button to download conversations as Markdown.
  • Control data sent: Adjust Context Depth in Settings (Summary or None sends less data).
  • Revoke access: Remove your API key from Settings at any time.

Third-Party Processing

Anthropic (San Francisco, CA, USA) processes your data as described in their Terms of Service. If your organization is subject to GDPR, review Anthropic's data processing terms before use.
